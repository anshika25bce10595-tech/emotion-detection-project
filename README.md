📍 Emotion Detection using Python

📌 Project Overview

This project is a real-time facial emotion detection system that identifies human emotions such as Happy, Sad, Angry, Surprise, etc. using a webcam. It uses computer vision and deep learning techniques.

🎯 Problem Statement

Computers cannot naturally understand human emotions. This project solves this problem by detecting emotions from facial expressions using AI.

🛠 Technologies Used

- Python
- OpenCV
- TensorFlow / Keras
- NumPy

✨ Features

- Detects face using webcam
- Predicts emotion in real-time
- Displays emotion on screen

▶️ How to Run the Project

1. Install required libraries:
   pip install opencv-python tensorflow numpy

2. Run the program:
   python emotion_detector.py

📁 Project Structure

- emotion_detector.py → Main program
- model/ → Contains trained model file
- haarcascade/ → Contains face detection XML file

💡 Output

The webcam opens and detects the face. The detected emotion is displayed on the screen in real-time.

⚠️ Note

Download the model file from:
https://github.com/omar178/Emotion-recognition
Place it inside the model folder before running the project.
